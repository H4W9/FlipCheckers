## Features

- Play against a friend or a CPU opponent
- Three CPU difficulty levels (CPU 1–3)
- Multi-jump captures — chain multiple jumps in a single turn
- Pieces promote to kings on reaching the back rank
- Mandatory jump rule (configurable in Settings)
- Save and resume games between sessions
- Haptic/vibration feedback

## Controls

| Input | Action |
|-------|--------|
| D-pad | Move cursor |
| OK | Select piece / confirm move |
| Back | Cancel selection / return to menu |

## Settings

| Option | Description |
|--------|-------------|
| Red | Set red player to Human or CPU (levels 1–3) |
| Black | Set black player to Human or CPU (levels 1–3) |
| Must Jump | Enforce mandatory capture rule (ON by default) |
| Vibro/Haptic | Enable or disable vibration feedback |
